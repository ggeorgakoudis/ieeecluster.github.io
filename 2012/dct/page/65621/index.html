Current Status and Trends¬†¬†¬†¬†¬†(Conference Room 307)
Speaker:Dr.Dhabaleswar K. (DK) Panda¬†¬†¬†¬†¬†¬†¬†¬†¬†¬†¬†
The Ohio State University
Abstract:

InfiniBand (IB) and High-Speed Ethernet (HSE) interconnects are generating a lot of excitement towards building next generation High Performance Computing (HPC) clusters and enterprise datacenters. This tutorial will provide an overview of these emerging interconnects, the features they offer, their current market standing, and their suitability for cluster computing. It will start with a brief overview of IB, HSE and their architectural features. An overview of the emerging OpenFabrics stack which encapsulates both IB and Ethernet in a unified manner, and hardware technologies such as Virtual Protocol Interconnect (VPI), RDMA over Converged Enhanced Ethernet (RoCE) that aim at converged hardware solutions will be presented. IB and HSE hardware/software solutions and the market trends will be highlighted. Finally, sample performance numbers highlighting the performance these technologies can achieve in different environments such as MPI, PGAS/UPC, Parallel File Systems, Memcached and Cloud Computing (Hadoop, HDFS and HBase) will be shown.

14:15-15:00Session I: Introduction to Dataflow Computing (45 mins)
Speaker: Prof. Haohuan Fu, Tsinghua University
Abstract:

Traditional control-flow oriented computing is facing severe challenges, such as the memory wall and the power wall. In contrast, dataflow computing, which is enabled by reconfigurable hardware devices, has demonstrated great potentials to provide more efficient computing solutions. In this talk, we will introduce the basic concepts of dataflow computing, the architecture of dataflow systems, and some existing application examples in exploration geophysics and banking.

15:00-15:30Coffee Break
15:30-16:15Session II: Dataflow Programming with MaxCompiler (45 mins)
Speaker: Prof. Xiaomeng Huang, Tsinghua University
Abstract:

Designing a customized dataflow solution for a specific application involves the same level of complexity to the process of designing an ASIC. Therefore, providing appropriate programming support for dataflow systems becomes a key issue for making it applicable to real applications. In this talk, we will introduce MaxCompiler, which significantly reduces the complexity of dataflow programming while keeping a high performance of the resulting solutions. We will go through a number of basic examples to demonstrate the basic design concept, and introduce the various features that MaxCompiler provides.

16:15-17:00Session III: The Acceleration Process - A Case Study (45 mins)
Speaker: Prof. Haohuan Fu, Tsinghua University
Abstract

In this talk, we will take a practical example to demonstrate the process for accelerating an application. We will cover various topics, from the profiling and analysis of the original software solution, to the different optimization techniques that we can take.

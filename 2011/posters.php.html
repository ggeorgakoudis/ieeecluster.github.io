IEEE Cluster 2011 Call for Posters

Scope

IEEE Cluster 2011 is the premier conference on all aspects of cluster computing for computational science. Cluster 2011 is hosted Austin, Texas, one of the leading technology cities in the U.S. Attendees will have access to the latest developments in cluster computing technologies and practices. Join the cluster computing community, and discuss new directions, opportunities, and ideas that will the future of cluster computing.

Cluster 2011 welcomes poster submissions on innovative work from researchers and practitioners in academia, government, and industry that describe original research and development efforts in cluster computing. Authors are invited to present their on-going or recently finished work related to cluster computing in the poster session of Cluster 2011. All accepted poster papers will be included in the Cluster 2011 proceedings. The Poster Session will be on Tuesday, September 27.

Topics include, but are not limited tothe following:

  • Systems Design and Configuration
    • Node and system architecture
    • Packaging, power and cooling
    • Visualization clusters and tiled displays
    • GPU clusters
    • Interconnect/memory architectures
    • Single system image clusters
    • Protocols, libraries, and interfaces
    • Lightweight communication protocols
    • Management & Reliability
    • Security and reliability
    • High availability solutions
    • Resource and job management
    • Scheduling, and load balancing
    • Administration and maintenance tools
    • Fault tolerance, checkpointing and recovery
    • Cost and performance implications of reliability
  • Tools, Systems Software, and Middleware
    • Software environments and tools
    • Virtualization and cloud computing
    • Performance evaluation, analysis and optimization
    • Grid and cloud computing technologies
  • Algorithms, Applications and Performance
    • Scalable algorithms
    • HPC Applications on GPUs
    • Data Intensive processing
    • Programming language and tools
    • Hybrid programming techniques (MPI+OpenMP, MPI+OpenCL, etc.)
    • Benchmarking & profiling tools
    • Performance prediction & modeling
  • Storage and File Systems
    • Storage cluster architectures
    • File systems and I/O libraries
    • Storage clusters
    • Storage clouds

Submission, Format, Publication

  • Submissions are accepted up through 23:59 on June 22, Austin time.
  • Poster articles must not exceed 4 pages.
  • Use the technical paper format as described in the Call for Papers at https://www.ieeecluster.org/papers.php.
  • Poster Submission will be available on May 17th.
  • All accepted poster papers will be included in the Cluster 2011 proceedings.

Sessions

The session schedule will be posted by June. Each poster author will be supplied with a foam core board on an easel and tacks. The boards will be 4830 inches (12275 cm) wide and 480 inches (122100 cm) high.

Authors will have early access to assemble their presentations.

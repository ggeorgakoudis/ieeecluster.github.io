Dynamic Reconfiguration in High-Speed Computer Clusters  

Dimiter Avresky
avresky@ece.neu.edu
 Northeastern University

Network component failures in high-speed local area networks or system
area networks can result in significant topological changes. In such
cases, a network reconfiguration algorithm is executed to restore the
connectivity. Most of the current network reconfiguration protocols
stop the user traffic to prevent cyclic dependencies in the routing
tables.

In this paper we specify NetRec, a novel dynamic reconfiguration
algorithm designed to increase the network availability in the presence
of faults. Unlike many other algorithms, NetRec is applicable for all
high-speed computer networks and is compatible with all modern routing
techniques, including wormhole routing system area networks.

It updates the routing tables asynchronously and doesn't require any
global knowledge about the network topology. Certain phases of NetRec
execute in parallel, thus reducing the reconfiguration time. The papers
provides examples of the algorithm operation and presents results from
validation of the algorithm in a distributed network simulation
testbed.

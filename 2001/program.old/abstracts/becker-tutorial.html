The Scyld Beowulf System

Donald Becker
becker@scyld.com
Scyld Computing Corporation

Beowulf systems are high performance computers constructed from commodity
hardware connected by a private internal network and running an open source
operating system infrastructure.

This tutorial will cover design, installation, deployment and running
Beowulf clusters. It will be focused on the Scyld Beowulf system, and
will include specific examples and a complete cluster installation
CD-ROM.

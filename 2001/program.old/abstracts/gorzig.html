CPPvm - Parallel Programming in C++  

Steffen Görzig 
steffen.goerzig@daimlerchrysler.com
Daimler-Chrysler, Germany

CPPvm is a C++ class library for message passing. It provides an
easy-to-use C++ interface to the parallel virtual machine software
PVM.

CPPvm is available on many architectures, from Windows up to several
UNIX derivatives. Although PVM can be used directly in C++ programs
due to its C-functions, it does not support C++ specific features.
CPPvm enlarges PVM with these C++ features like classes, inheritance,
overloaded operators, exception handling and streams. CPPvm also hides
some details of PVM from the user and thus makes it even easier to
write parallel programs.

This paper describes the concept of CPPvm, the main classes, the
transfer of C++ objects between processes, the use of distributed C++
objects and how to combine a heterogenous collection of computers
within CPPvm. It will also focus on the extension of CPPvm with user
defined classes. This allows, for example, to transform existing C++
programs into parallel programs running on a computer cluster.

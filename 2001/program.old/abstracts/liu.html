Cluster Trend: Data Speaking  

Peng Liu 
liup@tirc.cs.tsinghua.edu.cn
Zhihui Du
Tsinghua University  

Based on extensive analysis of the TOP500.Org and Clusters @ TOP500.Org
database, this paper aims to show the evolvement and current state of
cluster technology. The study reveals some amazing results: the cluster
is making so rapid a progress at an exponential speed, and along with
the steady increase of processors usage, the efficiency problem of
cluster is settled successfully. The paper also shows some important
statistics data from clusters, including the interconnection type, the
operating system type, and communication software type. Moreover, some
key features of current top clusters are provided as well.

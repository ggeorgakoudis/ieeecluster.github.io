Adaptive Runtime Partitioning of AMR Applications on Heterogeneous Clusters

Jyoti Batheja, Presenter
jbatheja@caip.rutgers.edu
Manish Parashar 
Shweta Sinha
Rutgers University 

This paper presents the design and evaluation of an 
adaptive system sensitive partitioning and load balancing
framework for distributed adaptive mesh refinement
applications on heterogenous and dynamic cluster
environments. The framework uses system capabilities and
current system state to select and tune appropriate
partitioning parameters (e.g. partitioning granularity,
load per processor) to maximize overall application
performance.

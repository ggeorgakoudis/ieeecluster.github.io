Hierarchical Cluster for Scalable Web Servers

Dongseung Kim
dkim@classic.korea.ac.kr
Jonghyuck Hong
Korea University

 Cluster architecture is widely used for the avail-ability and scalability of
web servers in these days. To increase the performance we have developed a
hierarchical architecture for web servers that can accommodate wide range of
connection requests by gradually expanding the nodes in proper levels in the
hierarchy. Especially, multiple dispatchers can distribute the incoming
client requests to vari-ous backend servers in the cluster. Experimental
results with two-level cluster with dynamic load balancing are included to
verify the idea.

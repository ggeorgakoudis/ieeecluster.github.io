SOVIA: A User-level Sockets Layer over Virtual Interface Architecture 

Jin-Soo Kim
jinsookim@etri.re.kr 
Kangho Kim 
Sung-In Jung
Electronics and Telecommunications Research Institute

The Virtual Interface Architecture (VIA) is an industry standard
user-level communication architecture for cluster or system area
networks. The VIA provides a protected, directly-accessible interface
to the network hardware, removing the operating system from the
critical communication path. In this paper, we present a user-level
Sockets layer over VIA, named SOVIA (Sockets Over VIA). Our goal is to
make the SOVIA layer as efficient as possible so that the performance
of native VIA can be delivered to user applications, while retaining
the portable Sockets semantics.

SOVIA realizes comparable performance to native VIA, showing the
minimum latency of 10.5usec and the peak bandwidth of 815Mbps on
Giganet's cLAN. The functional validity of SOVIA has been verified by
running FTP and RPC applications over SOVIA. Compared to the Giganet's
LANE driver which emulates TCP/IP inside the kernel, SOVIA shows 2.0 --
2.6 times higher bandwidth for FTP and 4.5 times lower latency for RPC
applications.

Future Directions for the Scyld Advanced 
Beowulf Management System

Donald Becker
becker@scyld.com
Scyld, Inc.

Scyld has introduced a new framework and set of tools for constructing
single system image Beowulf clusters. This talk will cover the
development direction of the system, including enhancements to the
security model, and the implications of extending the system model to
support multiple front-end "master". It will also cover other aspects
of the system, such as an integrated scheduler and enhanced libraries,
that are being enhanced to reduce the complexity of building and using
cluster applications..

An SCI-based Software VIA system for PC clustering  

Jeonghee Shin
jhshin@hyowon.pusan.ac.kr
Sang-Hwa Chung
Pusan National University 
Woo-Jong Hahn 
API Networks 

The performance of a PC cluster system is limited by the use of
traditional communication protocols, such as TCP/IP, because these
protocols are accompanied with significant software overheads. To
overcome the problem, systems based on user-level interface for
message-passing without intervention of kernel have been developed.
VIA(Virtual Interface Architecture) is one of the representative
user-level interfaces which provides low latency and high bandwidth.
In this paper, a software VIA system is implemented on an 8-node
SCI(Scalable Coherent Interface) network based PC cluster. The system
provides both message-passing and shared-memory programming
environments and shows the maximum bandwidth of 84MB/s and the minimum
latency of 8§Á in application level. The maximum speed-up of 7.6 was
obtained in executing the NAS parallel benchmark programs on the 8-node
SCI-based PC cluster. The system also shows better performance in
comparison with other comparable cluster systems in carrying out the
parallel benchmark programs.

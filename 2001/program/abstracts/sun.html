A Distributed Object Model for Solving Irregularly Structured Problems on Cluster

Ge Chen
gechen@csis.hku.hk
Yudong Sun 
ydsun@csis.hku.hk
Cho-Li Wang 
University of Hong Kong

This paper presents a distributed object model MOIDE for solving
irregularly structured problems on cluster. MOIDE is general-purpose
and architecture-independent, for developing and executing irregular
applications. The primary appeal of MOIDE is its flexible hierarchical
collaborative system structure that is adaptive to heterogeneous
architecture of a cluster. MOIDE integrates the object-oriented and
multithreaded methodologies to set up a unified computing environment.
Both the shared-data access and remote messaging are incorporated in a
two-layer communication mechanism for efficient inter-object
communication with the common communication interface. MOIDE supports
dynamic load balancing by its autonomous load scheduling technique. A
runtime support system is developed in Java and RMI that implements
MOIDE model as a platform-independent infrastructure to support
efficient execution of irregularly structured applications on cluster.
N-body, ray tracing, and conjugate gradient applications are developed
to illustrate the advantages of MOIDE model in supporting the
high-performance solutions for irregularly structured problems.

Programming with MPI on Clusters 

Rusty Lusk
Argonne National Laboratory

We discuss the current state of development for the key aspects of MPI
programming on clusters. These aspects are the evolution of the MPI
Standard itself, developments in cluster hardware and system software that
directly affect MPI implementations, and supporting software that
facilitates the use of MPI on scalable clusters. In each case we give a
brief background and summarize the current status.

GNET-1: Gigabit Ethernet Network Testbed

Yuetsu Kodama, et. al

GNET-1 is a fully programmable network testbed. It provides functions such as wide area network emulation, network instrumentation, trac shaping, and trac generation at gi- gabit Ethernet wire speeds by programming the core FPGA. GNET-1 is a powerful tool for developing network-aware grid software. It is also a network monitoring and trac-shaping tool that provides high-performance communication over wide area networks. This paper describes several sample uses of GNET-1 and presents its architecture.

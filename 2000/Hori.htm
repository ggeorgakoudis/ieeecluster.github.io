Biography

Atsushi Hori, Ph.D. is a Senior Researcher of Parallel and Distributed System Software Laboratory at Real World Computing Partnership, JAPAN. His current research interests include parallel operating system. He received B.S. and M.S. degrees in Electrical Engineering from Waseda University, and received Ph.D. from the University of Tokyo. He worked as a researcher in Mitsubishi Research Institute from 1981 to 1992.




Abstract

SCore: An Integrated Cluster System Software Package for High Performance Cluster Computing
Atsushi Hori
Senior Researcher
Real World Computing Partnership, Tsukuba, JAPAN

SCore Cluster System is an integrated software package for high performace clusters, and is used by worldwide users. SCore and the PM high performance communication library, which supports various networks available today, was designed simultaneously. As a result, SCore can provide not only high performance cluster computing environment, but also a number of cluster programming facilities in an integrated way.

The unique idea and features implemented in SCore will be addressed, including network trunking, network interleaving, gang-scheduling, checkpoint and restart, real-time monitoring of user programs, deadlock detection, debugging support, etc.
